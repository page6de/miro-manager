# miro-manager

A Browser Extension, that helps with Miro Board Password chaos


## Workflow:


#### Run on unsaved board

- after load: Check for title
  -> Save title for saved URL 


#### Run on saved board
 - Wait for page to load
 - Find PW Field
 - Enter PW
 - Submit PW Form

#### Popup:
 - List all saved Boards
 - Delete single savedBoards
 - Delete all
 - Enable Encryption?
